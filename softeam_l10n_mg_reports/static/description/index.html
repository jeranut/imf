Pour qui ?

Vous avez installé Comptabilité Malagasy (PCG 2005) sur votre Odoo et vous voulez maintenant produire les états financiers conformes au décret n°2004-272 du 18 février 2004 :

  • Déclaration TVA mensuelle ou trimestrielle
  • Bilan annuel pour l'assemblée générale et les banques
  • Compte de Résultat pour le suivi de performance
Ce module fournit les définitions account.report conformes au PCG 2005 Madagascar. Sur Odoo Enterprise, le viewer interactif (filtres, drill-down, export PDF/Excel, comparaisons de périodes) est disponible. Sur Community, les records sont chargés et lisibles via l'API.

3 rapports inclus

📊 Déclaration TVA Madagascar

Rapport tax_tags ventilé en 4 sections :

  • A. Chiffre d'affaires HT (taxable / export / exonéré)
  • B. TVA collectée 20 %
  • C. TVA déductible (B&S + immobilisations)
  • D. Calcul TVA à décaisser

Tags auto : +VENTE_BASE_20, +ACHAT_BS_TVA_20, +EXPORT_BASE, +EXO_VENTE_BASE…

📑 Bilan PCG 2005

Structure officielle Annexe I :

  • ACTIF non-courant : 20, 21, 22, 23, 26, 27, 28, 29, 133
  • ACTIF courant : 3, 41, 44D, 486, 50, 51+53+54+58
  • CAPITAUX PROPRES : 10, 11, 12
  • PASSIFS non courants : 13, 15, 16+17+18
  • PASSIFS courants : 40C, 42C, 43C, 44C, 481+487, 519

Engine account_codes — agrégation par préfixes PCG.

📈 Compte de Résultat par Nature

Selon PCG 2005 — analyse par nature obligatoire :

  • I. Production de l'exercice (70, 71, 72, 74, 75, 78)
  • II. Résultat opérationnel (Production − 60-65, 68)
  • III. Résultat financier (76 − 66)
  • IV. Résultat avant impôts (II + III)
  • V. Résultat net de l'exercice

Déclaration TVA — exemple

Avec les 4 factures démo incluses dans le module (scripts/seed_demo_invoices.py), voici ce que la déclaration affiche :

Ligne Description Montant (MGA)
A1CA local taxable 20 % (base HT) 1 000 000
A2CA exportation 0 % (base HT) 800 000
A4TOTAL Chiffre d'affaires 1 800 000
B1TVA collectée 20 % + 200 000
C2TVA déductible Biens & Services − 100 000
C4TVA déductible Immobilisations − 400 000
DTVA à décaisser (B − C) − 300 000 (crédit)

Le solde négatif est un crédit de TVA reportable sur le mois suivant (compte 4456 — Crédit de TVA à reporter).

Installation

  1. Installer d'abord Comptabilité Malagasy (PCG 2005) (softeam_l10n_mg) — ce module est sa dépendance.
  2. Charger le plan comptable Madagascar sur la société active (Settings → Companies → Madagascar - PCG 2005).
  3. Installer Rapports de Comptabilité Malagasy (PCG 2005) depuis l'Apps Store.
  4. Aller dans Comptabilité → Reporting pour accéder aux 3 nouveaux rapports.

Données de démo

Le module installe en mode démo 3 partenaires fictifs :

  • TANA SARL — Client local Antananarivo (test ventes locales TVA 20 %)
  • USA Trading Inc. — Client étranger (test ventes export 0 %)
  • FANIRY Sarl — Fournisseur Toamasina (test achats déductibles)

Pour générer aussi 4 factures démo (vente locale, export, achat B&S, achat immo), exécuter le script scripts/seed_demo_invoices.py :
